# 重量级 & Native Extension 依赖预安装文件 - 固定版本必须与 uv.lock 保持一致
#
# 用途：在独立 Docker 层中预安装这些包（--no-deps），使其与 uv.lock 细微变动解耦。
# 优先收录：① 体积大的 native extension 包（asyncpg/cryptography/pillow 等）
#           ② 核心重量级应用包（litellm/weasyprint/fastmcp）
# 效果：Layer B 的 uv sync 跳过这 ~25 个包，只需安装剩余的纯 Python 轻量包，速度快 2-3x。
#
# 维护：当 uv.lock 中以下包版本发生变化时，同步更新此文件。
# 重新生成命令（backend/ 目录执行）：
#   python3 - << 'EOF'
#   import tomllib, re
#   heavy = [
#     'litellm','openai','tiktoken','weasyprint','fastmcp',
#     'asyncpg','uvloop','cryptography','cffi','greenlet',
#     'aiohttp','aiosignal','frozenlist','multidict','yarl','propcache',
#     'pillow','pydantic-core','rpds-py','regex',
#     'grpcio','tokenizers','charset-normalizer',
#   ]
#   with open('uv.lock', 'rb') as f: lock = tomllib.load(f)
#   mp = {p['name']: p['version'] for p in lock.get('package', [])}
#   for pkg in heavy:
#       if pkg in mp: print(f'{pkg}=={mp[pkg]}')
#   EOF
